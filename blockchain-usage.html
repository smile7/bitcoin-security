Blockchain Usage

  • Sharing

    • The sharing applications are now more and more evolving. Such companies are Uber and AirBNB. Using blockchain technology, the world expands for other uses. For example, OpenBazar is creating a peer-to-peer eBay. The idea is to download the application and then everything you buy, you transact directly to the vendor and thus transaction fees are missed.

  • Crowdfunding

    • More and more people try their luck and invest time in the so called ICO - Inital Coin Offering. People have an idea, they work for it and then post their work in an ICO website. Whoever believes in the idea can invest in it by buying tokens which can become functional units of currency if the project’s funding goal is completed.

  • File Storage

    • Decentralizing file storage on the internet brings clear benefits. Distributing data throughout the network protects files from getting hacked or lost. Inter Planetary File System (IPFS) makes it easy to conceptualize how a distributed web might operate. Similar to the way a bittorrent moves data around the internet, IPFS gets rid of the need for centralized client-server relationships. Such an improvement is not only convenient. It’s a necessary upgrade to the web’s currently overloaded content-delivery systems.

  • You name it

    • This is the advantage of decentralized system. For example, a music app that lets people legally download music from the singer, processing direct payment.
