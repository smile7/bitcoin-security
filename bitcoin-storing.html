Bitcoin Storing

  • Online

    • Bitcoins are stored in a virtual wallet which contains Public Key, used to receive bitcoins - similar to the Bank Account Number, and a Private Key, used to identify the owner of the bitcoins that are about to be spent.

  • Offline

    • There are different ways of storing Bitcoins offline which are considered to be much more secure than the online alternative:

    1. Printed Wallet

      2. The digital wallet can be printed with both the public and the private key.

    2. Hardware Wallet

      3. In this case the data is stored in a microcontroller and there is absolutely no way to steal a wallet like on a normal computer. The microcontrollers cannot be compromised with viruses or other harmful software. Hardware stored bitcoins can also be used directly unlike the printed wallets.
